Deteksi Nomor Plat Kendaraan dengan Image Processing

Sistem Deteksi Nomor Plat Kendaraan berbasis OpenCV dengan Metode K-Nearest Neighbour

Tentang Projek

Sistem ini berfungsi untuk mendeteksi nomor plat kendaraan menggunakan image processing, sistem deteksi ini menggunakan bahasa pemrograman python dan library openCV. Sedangkan untuk proses deteksinya menggunakan metode K-Nearest Neighbour.

Prinsip Kerja

Kamera webcam akan mengambil gambar secara langsung pada kendaraan yng berada pada jangkauan kamera, data dari kamera akan diproses menggunakan program python pada komputer secara langsung pada program ini menggunakan library OpenCV.

Untuk tahap persiapannya hasil capture akan di lakukan proses pengolahan gambar terlebih dahulu agar memudahkan proses deteksi. Prosesnya antara lain efek negatif, biner, threshold, dan mengambil countur dari gambar, setelah didapatkan sample yang jelas maka akan dilakukan proses deteksi dengan mengambil nilai yang paling dekat.

Hasil

Data hasil deteksi diambil dengan jumlah karakter yang paling banyak berdasarkan jumlah karakter yang ada pada plat. hasil tersebut akan di tampilkan pada gambar dan menyimpan nomor plat yang terdeteksi pada file Excel.
